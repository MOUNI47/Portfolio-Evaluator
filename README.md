🚀 Portfolio Evaluator Web App

A full-stack web application that analyzes a developer’s GitHub profile and generates a detailed portfolio evaluation report with scores, insights, and visual breakdowns.

📌 Features
🔍 Analyze any GitHub user profile
📊 Generate portfolio score with breakdown
⭐ Top repositories with stars & forks
🌐 Language usage analysis
⚡ Caching for faster responses
💾 Store reports in MongoDB Atlas
📈 Ready for charts (Radar, Heatmap, etc.)
🛠️ Tech Stack
Frontend
React.js
Axios
Chart.js / Recharts
Backend
Node.js
Express.js
MongoDB Atlas
Mongoose
APIs
GitHub REST API
id
📊 Scoring Criteria

The portfolio score is calculated based on:

Number of repositories
Followers count
Repository activity
Stars & forks
Documentation (description)
Language diversity
